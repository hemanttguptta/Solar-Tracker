# Solar Tracking Device

## Overview

This project is a hardware-based solar tracker that automatically rotates a solar panel towards the direction of maximum sunlight using LDR sensors and an Arduino Uno.

## Features

- Automatic sunlight tracking
- Arduino Uno based
- LDR sensors
- Dual Servo Motors
- Low-cost prototype

## Components Used

- Arduino Uno R3
- LDR Sensors
- SG90 Servo Motors
- Solar Panel
- 10kΩ Resistors
- Jumper Wires
